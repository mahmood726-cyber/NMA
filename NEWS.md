# surroNMA 1.0.0

## Initial CRAN submission

### Major features

* Surrogate-assisted network meta-analysis with both Bayesian and frequentist engines
* Surrogate index (SI) training with multiple methods (OLS, Ridge, PCR, SuperLearner)
* Support for multivariate surrogates
* Inconsistency detection via node-splitting
* Class-specific and global surrogacy models
* Surrogate Threshold Effect (STE) computation
* Comprehensive ranking metrics (SUCRA, POTH, MID-adjusted)
* Automated HTML/PDF reporting
* Export to CINeMA format
* GUI interfaces (Tcl/Tk and gWidgets2)

### Bayesian features (requires CmdStan)

* MCMC and variational inference
* Random inconsistency models
* Posterior predictive checks

### Additional features

* Stress testing for surrogacy assumptions
* Risk of bias weighting
* Multi-arm trial adjustments
* Simulation functions for testing
